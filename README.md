# LuaLaTeX Thesis Template (APA 7th)

A clean, modern LaTeX thesis template built for **LuaLaTeX** with full support for **APA 7th Edition** citation style.



## How to Use
1. **Main File**: Compile `main.tex` using `latexmk -lualatex`.
2. **Configuration**: Edit `config.tex` for your name, title, and toggles (draft mode, ToC dots).
3. **Content**: Add your chapters in the `Chapters/` folder and include them in `main.tex`.
